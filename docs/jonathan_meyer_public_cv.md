## Profile

I build environmental-test capability for space hardware: the facilities, instrumentation, procedures, data visibility, and evidence flow that turn demanding campaigns into a repeatable engineering and production function.

My background combines six years leading environmental testing and qualification during startup-to-serial scale-up with a PhD foundation in cryogenic and vacuum instrumentation. The common thread is practical technical ownership across hardware, test infrastructure, data systems, and the evidence needed to make decisions.

This is a technical profile for collaborators, space-hardware teams, and serious conversations around environmental test, qualification, production readiness, and modern test-data systems.

## Public Demonstrator: Gossamer

Gossamer is an independent environmental-test demonstrator. It uses generic fixture data to show an operating model for campaign state, source provenance, DUT and test-bed data visibility, command authority, anomalies, requirements traceability, live evidence, and evidence reports.

The project exists to discuss test-system architecture and operator workflows without exposing employer data, lab procedures, real telemetry, proprietary protocols, or private identifiers.

GitHub: https://github.com/egidinas/gossamer
